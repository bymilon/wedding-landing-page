# Contributing

Thanks for helping improve Wedding Landing Page.

## Local Setup

```bash
bun install
bun run dev
```

Before opening a pull request, run:

```bash
bun run lint
bun run build
```

## Pull Requests

- Keep changes focused and explain the user-facing impact.
- Update README or TODO entries when behavior, setup, or roadmap scope changes.
- Do not commit secrets, guest data, API keys, or real RSVP submissions.
- For visual changes, include screenshots or a short before/after description.

## Project Direction

This project should remain an honest static wedding landing page template. Backend RSVP integrations, deploy adapters, and configuration improvements are welcome when they stay optional and easy to understand.
