# Security Policy

## Supported Versions

Security fixes target the `main` branch.

## Reporting A Vulnerability

Please report security issues privately by opening a GitHub security advisory when available, or by contacting the maintainer through the profile linked from the repository owner.

Do not include real guest data, API keys, or private event details in public issues.

## Current Security Notes

- The RSVP form is demo-only and does not persist data.
- No production backend, authentication, or payment flow is included.
- Any future RSVP backend should validate input server-side and protect guest data.
