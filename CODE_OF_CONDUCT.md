# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be kind and constructive.
- Respect differing experience levels and design tastes.
- Keep feedback focused on the work.

## Unacceptable Behavior

Harassment, insults, discrimination, sexualized language, and sustained disruption are not welcome.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block participants whose behavior harms the project community.
